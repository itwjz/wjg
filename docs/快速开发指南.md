环境要求\n- Node.js >= 14.0\n- MySQL >= 5.7\n- npm >= 6.0\n\n#### 2. 初始化项目\n\n```bash\n# 克隆项目\ngit clone https://github.com/itwjz/wjg.git\ncd wjg/backend\n\n# 安装依赖\nnpm install\n\n# 复制环境配置\ncp .env.example .env\n\n# 编辑 .env 文件，配置数据库和支付信息\nvim .env\n```\n\n#### 3. 数据库初始化\n\n```bash\n# 创建数据库\nmysql -u root -p\nCREATE DATABASE wjg_db DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;\n\n# 初始化表结构\nmysql -u root -p wjg_db < ../database/migrations/init.sql\n```\n\n#### 4. 启动开发服务器\n\n```bash\n# 开发模式（自动重启）\nnpm run dev\n\n# 生产模式\nnpm start\n```\n\n服务器将在 `http://localhost:3000` 启动\n\n---\n\n### 小程序开发环境搭建\n\n#### 1. 安装微信开发者工具\n- [官方下载](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)\n\n#### 2. 打开项目\n```bash\n# 在微信开发者工具中\n文件 → wx.setStorageSync('token', response.data.data.token);\n      }\n    });\n  }\n});\n```\n\n### 2. 购物车管理\n\n```javascript\n// 添加商品到购物车\nconst addToCart = (productId, quantity) => {\n checkout = async () => {\n  // 1. 创建订单\n  const orderRes = await API.order.create({\n    items: cartItems,\n    deliveryAddressId,\n    deliveryMethod,\n    totalAmount,\n    shippingFee\n  });\n\n  // 2. 创建支付\n  const paymentRes = await API.payment.create({\n    orderId: orderRes.data.orderId,\n    amount: orderRes.data.finalAmount,\n    paymentMethod: 'wechat'\n  });\n\n  // 3. 调起支付\n
