## 功能概述

### 核心目标
- 线下扫码快速购买微景观产品
- 支持安全在线支付
- 提供同城配送追踪
- 构建私域用户运营体系
- 数据分析支撑决策

### 产品定位
- **用户**: 微景观爱好者、办公室装饰采购、礼品赠送
- **场景**: 线下展台、门店、礼品区
- **特点**: 快速、便捷、同城可达

## 用户端功能

### 1. 首页模块

#### 1.1 扫码入口
- **功能**: 用户可直接扫描二维码进入对应商品页
- **流程**:
  1. 扫描二维码
  2. 解析参数 (store_id / product_id)
  3. 跳转到对应页面
  4. 无需重复登录

#### 1.2 首页展示
- **轮播图**: 活动/热销商品展示
- **分类导航**: 6-8个主要分类
- **热销商品**: 精选推荐 (6-8件)
- **新品上架**: 最新商品 (6-8件)
- **广告位**: 营销活动、优惠信息

#### 1.3 搜索功能
- **搜索框**: 关键词快速搜索
- **热搜词**: 实时热搜排行
- **搜索历史**: 用户搜索记录

### 2. 商品浏览模块

#### 2.2 商品详情页
- **商品信息**:

- **详情内容**:

### 3. 购物车模块

  - 修改数量 (增/减/删除)

### 4. 订单模块

#### 4.1 下单流程
```
商品确认 → 地址选择 → 配送选择 → 优惠选择 → 支付 → 完成
```

**第一步: 订单确认**
- 显示所选商品及数量
- 显示商品总价
- 支持修改数量

**第二步: 地址选择**
- 展示已保存地址列表
- 支持选择默认地址
- 支持添加新地址
- 新地址需填写完整信息

**第三步: 配送方式选择**
- **同城配送**: 选择送达时间
- **自取**: 选择自取时间和门店

**第四步: 优惠券选择**
- 展示可用优惠券列表
- 显示每张券的优惠金额
- 支持不选券

**第五步: 支付**
- 显示订单详情
- 显示应付金额
- 选择支付方式 (微信/支付宝)
- 调起支付

#### 4.2 支付结果
- **支付成功**: 显示订单号、配送信息、提示关注订单
- **支付失败**: 显示失败原因、重试选项
- **待支付**: 可在订单页查看并重新发起支付

## 业务流程

### 用户购买流程

```
扫描二维码
    ↓
进入小程序 → 检查登录 → 如未登录则自动登录
    ↓
查看商品 → 选择规格/数量
    ↓
加入购物车 or 立即购买
    ↓
确认订单 → 检查库存 → 库存不足提示
    ↓
选择收货地址 → 检查地址完整性
    ↓
选择配送方式 (同城/自取)
    ↓
选择优惠券 (可选)
    ↓
发起支付 → 调起微信/支付宝支付
    ↓
支付成功 → 订单生成 → 库存扣减
    ↓
发送订单通知 → 消息推送
    ↓
商家后台接收订单 → 提示发货
    ↓
商家确认发货 → 生成配送单
    ↓
配送员接单 → 取货
    ↓
用户实时追踪 → 收货
    ↓
订单完成 → 支持评价/售后
```

### 支付流程 (详细)

```
前端请求支付
    ↓
后端验证订单
    ├─ 检查用户登录
    ├─ 检查商品库存
    ├─ 验证优惠券有效性
    └─ 计算最终金额
    ↓
后端调用微信支付API
    ├─ 创建预支付订单
    ├─ 生成支付签名
    └─ 返回支付参数
    ↓
前端调用wx.requestPayment()
    ├─ 弹起微信支付
    └─ 用户确认支付
    ↓
微信支付完成
    ├─ 同步回调 → 前端显示结果
    └─ 异步回调 → 后端处理业务
    ↓
后端处理异步回调
    ├─ 验证签名
    ├─ 生成订单
    ├─ 扣减库存
    ├─ 更新会员积分
    ├─ 发送通知
    └─ 返回成功确认
    ↓
订单进入待发货状态
```

## 数据字典

### 订单状态

| 状态码 | 状态名 | 说明 |
|------|--------|------|
| pending | 待支付 | 订单已生成，等待用户支付 |
| paid | 待发货 | 订单已支付，等待商家发货 |
| shipped | 配送中 | 订单已发货，配送员配送中 |
| delivered | 已完成 | 订单已送达，用户已签收 |
| cancelled | 已取消 | 订单已取消 |

### 配送状态

| 状态码 | 状态名 | 说明 |
|------|--------|------|
| pending | 待配送 | 等待配送员接单 |
| picked | 已取货 | 配送员已取货 |
| delivering | 配送中 | 配送员配送中 |
| delivered | 已送达 | 用户已签收 |
